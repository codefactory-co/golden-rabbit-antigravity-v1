# 요즘 안티그래비티 - 실전 프로젝트 모음

이 레포지토리는 **"요즘 안티그래비티"** 과정에서 다루는 실전 프로젝트들의 소스 코드를 담고 있습니다.  
각 프로젝트는 최신 웹 기술 스택(Next.js, Supabase, TypeScript 등)을 기반으로 하며, 단계별로 핵심 아키텍처와 개발 방법론을 학습할 수 있도록 구성되어 있습니다.

## 📂 프로젝트 목록

각 디렉토리에는 해당 프로젝트의 전체 소스 코드와 설정 파일이 포함되어 있습니다.

### 기초 및 방법론 (Fundamentals)

| 프로젝트 | 경로 | 설명 |
| :--- | :--- | :--- |
| **Project 4: TDD** | [`4/tdd`](./4/tdd) | 테스트 주도 개발(Test Driven Development)을 통한 견고한 코드 작성법 학습 |
| **Project 5: Clean Architecture** | [`5/clean architecture`](./5/clean%20architecture) | 유지보수성과 확장성을 극대화하는 클린 아키텍처 설계 패턴 실습 |

### 실전 애플리케이션 (Applications)

## 🚀 시작하기 (Getting Started)

각 프로젝트 폴더로 이동하여 의존성을 설치하고 개발 서버를 실행하세요.

```bash
# 예시: Project 11 실행
cd 11/saas
npm install
npm run dev
```

## 🛠 주요 기술 스택

- **Framework**: Next.js (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Backend/DB**: Supabase
- **Deployment**: Vercel
