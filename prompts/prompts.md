# Antigravity Projects Prompts

## Chapter 08: SaaS Landing Page

### 08-01: Landing Page Design

‘WriteFlow’라는 AI 글쓰기 도우미 SaaS의 랜딩 페이지를 만들어 줘. 모던하고 깔끔한 디자인, 인디고/보라색 계열 색상, 모바일 반응형으로 만들어 줘.

#### 1. 히어로 섹션
- AI와 함께 더 빠르게 더 잘 쓰세요
- 하위 헤드라인: "아이디어를 몇 초 만에 완성된 콘텐츠로 바꿔보세요"
- 버튼 : "무료 체험 시작" (주요), "데모 보기" (보조)
- 인디고에서 보라색으로 이어지는 은은한 그라데이션 배경

#### 2. 기능 소개 (3×2 그리드)
- AI 글쓰기 어시스턴트: "클릭 한 번으로 콘텐츠 생성"
- 언제나 완벽한 문법
- 브랜드 목소리에 맞게 조절
- 50개 이상 언어로 작성합니다.
- 100개 이상 바로 쓸 수 있는 템플릿
- 실시간으로 함께 작업합니다.
- 각 카드에 아이콘, 제목, 설명을 포함합니다. 호버 효과를 추가합니다.

#### 3. 가격 테이블 (3개 플랜)
- Free ($0): 월 5개 문서, 기본 문법 검사, 이메일 지원
- Pro ($19, "가장 인기" 배지): 무제한 문서, 고급 AI, 우선 지원, API
- Enterprise ($49): Pro 전체 + 맞춤 AI 학습, 전담 매니저, SSO
- Pro 플랜 시각적 강조, 월간/연간 토글 포함

#### 4. FAQ (아코디언)
- 무료 체험은 어떻게 진행되나요?
- 언제든지 구독을 취소할 수 있나요?
- 제 데이터는 안전한가요?
- 환불이 가능합니까?
- 어떤 결제 수단을 지원하나요?
- 플러스/마이너스 아이콘, 부드러운 애니메이션

#### 5. CTA 섹션
- 헤드라인: "글쓰기를 혁신할 준비가 되셨나요?"
- 10,000명 이상인 작가와 함께하세요
- 무료 체험 시작 버튼, 신용카드 필요 없음 텍스트
- 히어로와 같은 그라데이션 배경

## Chapter 09: Blog Platform

### 09-01: Blog UI Design

개발자 블로그 플랫폼의 전체 UI를 디자인해줘.

**[디자인 컨셉]**
- 미니멀하고 깔끔한 개발자 블로그 스타일
- 다크 모드 기본, 라이트 모드 지원
- 마크다운 콘텐츠에 최적화된 타이포그래피

**[페이지 목록]**

1. **홈페이지 (/)**
   - 헤더: 로고, 검색바, 로그인/회원가입 버튼
   - 게시글 카드 그리드(2열 또는 3열)
   - 페이지네이션

2. **게시글 상세 페이지 (/posts/[id])**
   - 제목, 작성자 정보, 작성일
   - 마크다운 본문
   - 좋아요 버튼
   - 댓글 섹션

3. **글쓰기 페이지 (/posts/new)**
   - 제목 입력
   - 마크다운 에디터(좌: 에디터, 우: 미리보기)
   - 발행 / 임시 저장 버튼

4. **로그인/회원가입 페이지**
   - 이메일, 비밀번호 입력
   - 소셜 로그인 버튼 (구글, 깃허브)

## Chapter 10: Shopping Review Analytics Chatbot

### 10-01: Chatbot Interface Design

쇼핑 리뷰 분석 챗봇 웹 앱의 전체 UI를 만들어줘.

**[프로젝트 개요]**
- 상품 리뷰를 AI가 분석해서 사용자 질문에 답변하는 챗봇
- Next.js 16 + TypeScript + Tailwind CSS + shadcn/ui
- 라이트 모드 기본, 모던하고 깔끔한 디자인

**[필요한 페이지 / 컴포넌트]**

1. **메인 채팅 페이지 (/page.tsx)**
   - 상단: 헤더 (로고, 상품명 "프리미엄 무선 이어폰 Pro", 설정 버튼)
   - 중앙 : 채팅 메시지 목록 (오른쪽 정렬, 파란색 배경)
     - AI 메시지: 왼쪽 정렬, 회색 배경, 아바타 아이콘
     - AI 메시지 하단에 "참고한 리뷰" 섹션(접을 수 있는 카드 형태)
   - 하단: 입력창 + 전송 버튼 (고정)
   - 로딩 중일 때 타이핑 인디케이터 표시

2. **출처 리뷰 카드 (SourceCard)**
   - 별점 (5점 만점, 별 아이콘)
   - 작성자명
   - 리뷰 내용(2줄까지만 표시, 더 보기)
   - 관련도 점수 표시

3. **리뷰 요약 대시보드 카드 (ReviewSummaryCard)**
   - 상품명, 총 리뷰 수
   - 평균 별점 (큰 숫자 + 별 아이콘)
   - 감정 분포 바(긍정: 초록, 중립: 회색, 부정: 빨강)
   - 장점/단점 리스트 (2열 그리드, ThumbsUp/ThumbsDown 아이콘)
   - 주요 키워드 태그들

4. **사이드바(선택적으로 표시)**
   - 이전 대화 목록
   - 새 대화 시작 버튼
   - 상품 선택 드롭다운

5. **빈 상태 / 웰컴 메시지**
   - 처음 접속 시 보여줄 환영 메시지
   - 운동할 때 써도 돼요?, 배터리 오래 가나요?, 통화 품질은?

**[디자인 요구사항]**
- 색상 : 라이트 모드 기본(white/gray-50 배경, white 카드, gray-100 보더)
- 강조색: blue-500 (사용자 메시지, 버튼)
- AI 메시지: gray-100 배경, 검은색 텍스트
- 아이콘: lucide-react 사용
- 반응형: 모바일에서도 잘 보이게
- 애니메이션: 메시지 등장 시 fade-in, 로딩 시 pulse

## Chapter 11: Shopping Mall Dashboard

### 11-01: Dashboard Design

#### 1. 메인 대시보드
온라인 쇼핑몰 관리자용 대시보드 메인 화면을 만들어줘.
- **상단 카트 (4개 핵심 지표)**:
    - 오늘 매출 (금액, 전일 대비 퍼센트 변화)
    - 신규 주문 (건수, 전일 대비 변화)
    - 신규 고객 (명수, 전일 대비 변화)
    - 재고 부족 상품 (개수, 경고 아이콘)
- **중간 차트**:
    - 왼쪽: 주간 매출 추이 (선 그래프, 월~일)
    - 오른쪽: 카테고리별 판매 비율 (파이 차트)
- **하단 테이블**:
    - 최근 주문 목록 (주문번호, 고객명, 상품, 금액, 상태, 시간)
    - 5개 행 정도, 오른쪽 상단 "전체 보기" 링크
- **사이드바**: 대시보드, 상품관리, 주문관리, 고객관리, 분석, 설정
- **스타일**: 깔끔한 흰색 배경에 파란색 포인트. 한국어.

#### 2. 상품 목록 페이지
쇼핑몰 관리자가 상품을 관리하는 페이지를 만들어줘.
- **상단**: 제목 "상품 관리", 파란색 "상품 등록" 버튼
- **검색/필터**: 검색창(상품명), 카테고리 드롭다운, 재고 상태 드롭다운
- **테이블**: 체크 박스, 이미지 썸네일, 상품명, 카테고리, 가격, 재고, 상태(재고 부족 빨간색), 등록일, 수정/삭제 버튼
- **하단**: 페이지네이션, 선택된 항목 일괄수정/삭제 버튼
- **사이드바**: 포함. 한국어.

#### 3. 상품 등록/수정 폼
쇼핑몰에 새 상품을 등록하는 폼 페이지를 만들어줘.
- **상단**: 제목 "상품 등록/수정", 취소/저장 버튼
- **기본 정보**: 상품명(필수), 상품 설명(텍스트 영역), 카테고리, 가격, 재고 수량
- **이미지**: 업로드 영역(드래그 앤 드롭), 미리보기, 삭제 버튼
- **AI 도우미**: "AI로 상품 설명 생성", "SEO 태그 추천" 버튼
- **사이드바**: 포함. 한국어.

#### 4. 주문 목록 페이지
쇼핑몰 주문을 관리하는 페이지를 만들어줘.
- **상단**: 제목 "주문 관리"
- **탭 버튼**: 전체 | 결제대기 | 결제완료 | 배송중 | 배송완료 (선택됨 강조)
- **검색/필터**: 주문번호/고객명 검색, 날짜 범위 선택
- **테이블**: 주문번호, 고객명, 상품요약, 결제금액, 주문상태(배지), 주문일시
- **하단**: "오늘 처리해야 할 주문: 15건" 안내
- **사이드바**: 포함. 한국어.

#### 5. 주문 상세 페이지
주문 상세 정보를 보여주는 페이지를 만들어줘.
- **상단**: "주문 #1234", 이전/목록/다음 버튼
- **진행바**: 결제대기 → 결제완료 → 상품준비 → 배송중 → 배송완료
- **정보 카드**: 주문 정보, 고객 정보, 배송 정보(운송장 입력 포함)
- **상품 카드**: 이미지, 상품명, 단가, 수량, 소계, 합계 정보
- **하단**: 주문 취소, 상태 변경 저장
- **한국어**.

#### 6. 고객 목록 페이지
쇼핑몰 고객을 관리하는 페이지를 만들어줘.
